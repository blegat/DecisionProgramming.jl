analysis.jl

DecisionProgramming.ActivePaths — Type

Interface for iterating over active paths given influence diagram and decision strategy.

  1. Initialize path s of length n
  2. Fill chance states s[C] by generating subpaths paths(G.C)
  3. Fill decision states s[D] by decision strategy Z and path s

Examples

for s in ActivePaths(G, Z)
    ...
end

random.jl

DecisionProgramming.InfluenceDiagram — Method

Create random influence diagram of given size.

Examples

import Random
rng = MersenneTwister(1)
n_C, n_D, n_V = 4, 2, 1  # Number of chance, decision and value states
n_A = 2  # Upper limit on information set size for any node
num_states = [2, 3]  # Set of possible number of states
G = InfluenceDiagram(rng, n_C, n_D, n_V, n_A, num_states)
